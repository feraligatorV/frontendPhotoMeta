# prerequisites
 1. Node.js
 2. React.js
 2. npm

# getting started
 1. npm install
 2. npm run build:dev for make a build with dev
 3. npm start dev for run build dev
 4. npm run build:prod for make a build with prod
 5. npm start prod for run build prod
